### Installation

Clone the repository:
```bash
git clone https://github.com/your-username/calculator-app.git
```
Navigate to the project directory:
```bash
cd ~./DFA_Calculator
```
Install dependencies:
```bash
pip install tkintertable
```

###Usage
```bash
python Calculator.py
```
