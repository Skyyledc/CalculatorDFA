## Prerequisites

- Python 3.x
- Tkinter library

## Installation

Clone the repository:
```bash
git clone https://github.com/Skyyledc/CalculatorDFA.git
```
Navigate to the project directory:
```bash
cd ~./DFA_Calculator
```
Install dependencies:
```bash
pip install tkintertable
```

## Usage
```bash
python Calculator.py
```
